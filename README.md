## 📖 What is Typora_Claude-Like_Theme?

This is a theme designed for Typora, a popular markdown editor. It aims to provide a calm, focused reading and writing experience, inspired by Claude-like styles. It works well for Chinese and English writing. The theme offers two versions: a light and a dark mode. Its design helps you keep attention on your content, whether you write notes, reports, or tech documents.

## 📂 How to Install the Theme on Windows

Follow these steps to install the theme and start using it in Typora:

1. **Open Typora**  
   Launch the Typora application on your Windows computer.

2. **Open the Theme Folder**  
   In Typora, go to the menu:  
   `Settings / Preferences -> Appearance -> Open Theme Folder`  

   This opens the folder where Typora stores its theme files.

3. **Copy Theme Files**  
   Copy the downloaded files `claude-like.css` and `claude-like-dark.css` into the theme folder you just opened.

4. **Restart Typora**  
   Close Typora and open it again. Restarting allows the new themes to load properly.

5. **Choose Your Theme**  
   In Typora, go to:  
   `Settings / Preferences -> Appearance -> Theme`  
   Select `Claude-like` or `Claude-like Dark` from the available themes.

## 🛠 Recommended Settings for Windows

For the best visual experience, change the window style in Typora:

1. Go to `Settings / Preferences -> Appearance`.  
2. Find the option **Window Style**.  
3. Set it to **Unified** (一体化).  
4. Restart Typora one more time.

This setting works best with the Claude-Like theme, especially on Windows machines.

- **Wide spacing** that helps your eyes move easily over long text.

The theme focuses on a calm reading atmosphere that does not draw attention away from your writing.
